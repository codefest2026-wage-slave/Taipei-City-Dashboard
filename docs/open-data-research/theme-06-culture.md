# 主題06：文化共融 — 開放資料研究

> 調研日期：2026-04-21
> 資料來源：data.taipei, data.ntpc.gov.tw, data.gov.tw
> 調研目標：找出未在官方儀表板實作的文化共融相關資料集

## 已跳過（官方已實作）

- 觀光景點分布（data.taipei 已實作）
- 觀光景點直播
- 旅館統計
- 友善店家資源分布（行動裝置充電 / 英文友善 / 自行車友善 / 素食 / 穆斯林友善 / 無障礙友善 / 月經友善 / 哺集乳友善 / 友善廁所）
- 捷運站乘客流動量（商圈角度）
- 公廁分布
- AED 資源設置分布概況

> **調研說明**：data.taipei 的搜尋 API 於本次調研期間持續回傳 500 錯誤，改以下列方式取得資料：
> 1. 直接訪問 data.taipei 各機關頁面並攔截 API 回應
> 2. 從 data.gov.tw（政府資料開放平臺）搜尋台北市文化相關資料集
> 3. 使用 data.ntpc.gov.tw 的 `/api/v1/dataset.search` API 搜尋新北市資料集
