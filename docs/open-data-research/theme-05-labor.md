# 主題05：勞動福祉 — 開放資料研究

> 調研日期：2026-04-21
> 資料來源：data.taipei, data.ntpc.gov.tw
> 調研目標：找出未在官方儀表板實作的勞動福祉相關資料集

## 已跳過（官方已實作）

- 育嬰假性別比、幼兒人數統計、育兒支持資源、哺集乳室分佈、生育補助合約醫院
- 高齡就業人口之年增結構、扶養比及老化指數、全市年齡分區、長照指標
- 臺北市青年創業融資貸款累計件數統計、臺北市青年創業融資貸款核准狀態
- 各行政區性別人口分布、各行政區年齡層人口分布
