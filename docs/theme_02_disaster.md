# 民生示警公開資料平台（NCDR CAP）

## 即時示警 RSS / CAP 資料

- **來源平台**: 民生示警公開資料平台（國家災害防救科技中心 NCDR）
- **API端點**:
  - RSS（免金鑰）：`https://alerts.ncdr.nat.gov.tw/RssFileData.ashx`
  - 示警訊息清單 API：`https://alerts.ncdr.nat.gov.tw/api/v1/alerts/dataListtore`
  - 發布單位/示警名稱查詢：`https://alerts.ncdr.nat.gov.tw/api/v1/alerts/dataListet`
  - CAP XML 單筆：`https://alerts.ncdr.nat.gov.tw/api/v1/alerts/capData?capid={capid}`
- **相關主題**: 韌性防災（全類型警報）
- **雙北支援**: ✅ 含地方政府台北市、新北市發布的示警
- **資料格式**: RSS（Atom/XML）；API 回傳 JSON；單筆 CAP XML
- **更新頻率**: 即時（示警發布時即更新）
- **主要欄位（CAP 格式）**:
  - `identifier`（capid，唯一識別）
  - `sender`（發布單位）
  - `sent`（發布時間）
  - `status`（Actual/Exercise/Test）
  - `msgType`（Alert/Update/Cancel）
  - `scope`（Public/Restricted）
  - `info.event`（示警類型名稱）
  - `info.urgency/severity/certainty`（緊急度/嚴重度/確定性）
  - `info.headline`（標題）
  - `info.description`（詳細說明）
  - `info.area.areaDesc`（受影響區域描述）
  - `info.area.geocode`（行政區代碼）
  - `info.area.polygon`（影響範圍多邊形座標）
- **示警類型涵蓋**:
  - 淹水、河川高水位、水庫放流、枯旱預警（水利署）
  - 地震速報（中央氣象署）
  - 颱風、強風、大雨（中央氣象署）
  - 土石流（農業部水保署）
  - 海嘯（中央氣象署）
  - 空品不良（環保署）
  - 道路管制、橋梁封閉（公路局/縣市交通局）
- **備註**:
  - 查詢 API 步驟：先查 `dataListet` 取得發布單位/示警代碼 → 查 `dataListtore` 取即時清單與 capid → 查 `capData` 取完整 CAP 內容
  - RSS 可直接訂閱，無需 API 金鑰，適合快速整合
  - 地方政府（台北市、新北市）亦有自行發布示警入此平台

# 重要注意事項

1. **TDX 認證**: 使用 OIDC Client Credentials 取得 Bearer Token，免費方案：50次/日（訪客），會員方案依等級有更高限制
2. **OData 語法**: 所有 TDX API 支援 `$filter`, `$select`, `$orderby`, `$top`, `$skip`, `$format` 等 OData 查詢參數
3. **民生公共物聯網遷移**: 原 `sta.ci.taiwan.gov.tw` 已改為 `sta.colife.org.tw`，預計服務至 2026-12-01
4. **雙北公車 V2**: 新北市 V2 資料由雙北雲系統介接，部分欄位（StopStatus、NextBusTime 等）可能為空值
5. **歷史資料**: TDX 路況歷史資料提供 2021-07 起至前一天（每日更新），最多查詢7日
6. **授權宣告**: 介接 TDX 資料後，應用服務需揭露「資料介接交通部TDX平臺」聲明
