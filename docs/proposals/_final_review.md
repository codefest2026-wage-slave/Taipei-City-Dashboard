## 跨提案最致命共同盲點 (團隊章程必寫)

1. **AI 責任邊界**: 醫療/補助/法律建議/族群代理全部踩線. **章程**: 任何 AI 輸出前面必加 disclaimer + 標"AI為初篩, 非最終認定" + 用 deterministic 規則白名單兜底.
2. **雙北對稱包裝**: ID 列表偽裝雙北平等, 實際新北側資料缺. **章程**: 把缺口本身做成 "資料覆蓋透明聲明" → 用透明取代隱瞞 → 評審反而加分.
3. **32 hr 樂觀**: MVP scope 普遍超載. **章程**: Day1 ship 1 雙北組件 + 1 地圖. 砍 30% feature. 寧縮不爛.
4. **30 RPM 並發保險**: AI 用法多預期高頻互動. **章程**: 前端 cache + batch 預生成 + LLM down 時 template fallback (3 層).
5. **大螢幕無人模式**: 6 提案皆 mock. **章程**: 30s 自動輪播 + 預錄 persona 巡演 + 字級 ≥24px + 16:9 主視覺 70%.
6. **行動入口都是 hard-code**: 「點此通報」全是 mock. **章程**: 至少 1 個提案有真實 deep-link (LINE/政府 API/1999).
