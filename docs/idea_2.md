# Idea：韌性防災即時監測儀表板

## 主題
韌性防災——整合氣象、水情與災害風險資料，建立即時監測與應變決策機制，強化城市對極端氣候事件之韌性。

---

## UI 組件設計（4 個）

### 組件 1｜即時災害風險地圖（地圖圖層）
**類型**：地圖（Leaflet / MapLibre GL）

展示臺北市與新北市的多層次空間風險資訊，使用者可自由切換圖層。

| 圖層名稱 | 資料來源 |
|---------|---------|
| 雨量站分布 | 臺北市水利處雨量站 |
| 土石流觀測點 | 臺北市工務局土石流觀測影像 CCTV |
| 防災公園 | 臺北市防災公園 |
| 避難收容處所 | 臺北市可供避難收容處所一覽表、新北市避難收容處所一覽表 |
| 新北各區防災地圖 | 新北市各區防災地圖 |

**互動行為**：點擊站點顯示即時數值（雨量、警戒等級）；圖層切換面板於右上角；顏色編碼反映當前警戒狀態（綠/黃/紅）。

---

### 組件 2｜水情與雨量趨勢監測
**類型**：折線圖 + 指標卡片

整合水庫水情與降雨歷史趨勢，提供管理者快速判斷水資源與洪災風險。

**資料來源**：
- 臺北翡翠水庫即時水情資料
- 臺北市水利處歷史日雨量
- 臺北翡翠水庫水文氣象月報表

**呈現**：
- 上方 3 個指標卡片：當前水庫水位 / 今日累積雨量 / 本月均雨量
- 下方雙軸折線圖：左軸水位（m）、右軸降雨量（mm），時間範圍可選 7日 / 30日 / 90日
- 各警戒閾值以橫線標示（黃色警戒 / 紅色警戒）

---

### 組件 3｜多災種風險警戒等級儀表板
**類型**：儀表板（Gauge + 風險矩陣）

整合不同災害類型的雨量警戒值，一覽各類型目前風險程度。

**資料來源**：
- 臺北市土石流雨量警戒值
- 臺北市列管邊坡紅黃色雨量警戒值
- 臺北市老舊聚落紅黃色雨量警戒值

**呈現**：
- 3 個半圓 Gauge：土石流風險 / 邊坡崩塌風險 / 老舊聚落積水風險
- 每個 Gauge 對應目前累積雨量與警戒閾值的比例，顏色三段（安全 / 警戒 / 危險）
- 底部顯示各警戒區域的地點名稱清單，點擊可跳轉至地圖組件對應位置

---

### 組件 4｜避難設施與社區防災資源查詢
**類型**：搜尋列表 + 卡片

提供市民快速查詢就近避難設施，強化自主應變能力。

**資料來源**：
- 臺北市可供避難收容處所一覽表（含 114 年最新版）
- 臺北市各行政區各里簡易疏散避難地圖資訊（松山、大同、北投、萬華、信義、中正、文山、大安、中山、南港、內湖、士林等 12 區）
- 新北市避難收容處所一覽表
- 新北市避難收容處所優先排序自主檢核表
- 臺北市山坡地自主防災社區推動列表
- 臺北市水患自主防災韌性社區一覽

**呈現**：
- 頂部：行政區 / 里 下拉選單篩選，或輸入地址關鍵字搜尋
- 結果以卡片列表呈現，每卡顯示：設施名稱、地址、可容納人數、距離（若取得定位）
- 底部標籤頁切換：「避難設施」 vs 「防災社區」
- 點擊任一設施卡片，地圖組件（組件 1）自動飛到該位置並高亮顯示

## 組件互動關係

```
[組件3 風險儀表板] ──點擊警戒區域──→ [組件1 地圖] 飛至該位置
[組件2 水情趨勢]  ──超過警戒值時──→ [組件3] 自動更新狀態
[組件4 避難查詢]  ──點擊設施卡片──→ [組件1 地圖] 高亮定位
```

---

## 技術建議

- **地圖**：Leaflet.js 搭配 GeoJSON，或 MapLibre GL 支援向量圖層
- **圖表**：ECharts（支援 Gauge + 折線雙軸）或 Plotly
- **資料更新**：水庫水情可設每 30 分鐘輪詢；雨量站建議每 10 分鐘更新
- **框架**：Gradio (gr.HTML) 或 React + Tailwind CSS
