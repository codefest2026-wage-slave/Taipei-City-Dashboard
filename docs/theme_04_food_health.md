## 使用建議（黑客松視角）

### 食安健康儀表板
- **違規食品地圖**：食藥署違規食品標示 + 縣市座標 → 台北市/新北市熱點地圖
- **疾病趨勢折線圖**：疾管署類流感/腸病毒每週急診人次 → 雙北趨勢比較
- **登革熱即時警示**：疾管署登革熱聚集區域（每日更新 GeoJSON）
- **農藥殘留合格率**：農業部稻米農藥殘留 + 食藥署標準 → 合格率統計

### 文化共融儀表板
- **藝文活動地圖**：文化部全類型活動 API（每日更新）→ 雙北活動密度熱圖
- **博物館分布**：公私立博物館 JSON → 地圖標記 + 類型篩選
- **文化活動日曆**：startDate/endDate 欄位 → 時間軸呈現
