# Idea：食安健康儀表板

## 主題
食安健康 — 整合雙北食品安全抽驗、業者分布、健康服務資源，透過風險地圖與指標趨勢，協助政府強化稽查決策並提升市民自我健康管理能力。

---

## UI 組件設計（共 4 個）

### 組件 1｜食安風險地圖（地圖圖層）
**功能：** 以互動地圖呈現雙北食品業者分布，疊加食品抽驗不合格熱點，讓稽查人員快速識別高風險區域。

**互動設計：**
- 圓點標記：食品工廠 / 業者位置（可依類別篩選）
- 熱力圖層：依抽驗不合格次數產生熱區
- 點擊標記：顯示業者名稱、最近抽驗時間、合格/不合格記錄
- 時間軸篩選器：查看特定季度的違規分布

**使用資料集：**

|--------|------|-----|

| 新北市食品抽驗資料 | 新北市 | `https://data.ntpc.gov.tw/api/datasets/078cb722-15ac-4e1e-b541-e75bfe0aa440/json` |

### 組件 2｜食品抽驗合格率趨勢儀表板
**功能：** 以折線圖呈現雙北市售食品抽驗合格率的歷年趨勢，並分拆標示合格率與抽驗合格率兩項指標，協助政府評估管制成效。

**互動設計：**
- 雙軸折線圖：台北 vs 新北合格率並排對比
- 切換卡：標示合格率 / 抽驗合格率
- 數字卡片：顯示最新一期合格率、較上期變化百分比
- 下拉選單：依食品類別（肉品、飲料、醬料等）篩選

**使用資料集：**
| 資料集 | 來源 | API |
|--------|------|-----|
| 市售食品標示合格率 | 新北市 | `https://data.ntpc.gov.tw/api/datasets/b57c14e9-593c-4416-8edd-d1ac8cdceb48/json` |
| 市售食品抽驗合格率 | 新北市 | `https://data.ntpc.gov.tw/api/datasets/d2d69f7e-e283-406e-859b-e9e5dc98ac50/json` |
| 臺北市食品衛生管理查驗工作 | 台北市 | `https://tsis.dbas.gov.taipei/statis/webMain.aspx?sys=220&ymf=9500&kind=21&type=0&funid=a05032002&cycle=4&outmode=12&compmode=0&outkind=1&deflst=2&nzo=1` |

### 組件 3｜健康服務資源地圖（地圖圖層）
**功能：** 展示雙北健康服務中心、健康守護站、老人健康檢查特約醫事機構的空間分布，協助市民就近找到健康資源。

**互動設計：**
- 分層標記：健康服務中心（藍）/ 健康守護站（綠）/ 長者檢查機構（橘）
- 側邊資訊卡：點擊標記後顯示服務項目、開放時間、聯絡資訊
- 我的位置：定位後自動顯示半徑 1km 內資源
- 圖層開關：可個別開啟/關閉各類型資源

### 組件 4｜餐飲衛生評核與違規廣告統計
**功能：** 以環狀圖與長條圖呈現通過餐飲衛生管理分級評核的業者數量，並對照違規食品廣告的裁罰統計，協助民眾辨別優良業者、避開違規品牌。

**互動設計：**
- 評核業者排行：依評核等級（A/B/C）分類顯示業者數量環狀圖
- 違規廣告長條圖：依違規類型（食品/藥品/化妝品）分類，可按季度切換
- 業者搜尋框：輸入店名查詢是否通過衛生評核
- 摘要文字卡：本季新增評核業者數 / 新增違規裁罰件數
