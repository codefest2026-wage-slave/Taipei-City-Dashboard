## 四、地圖交叉比對（/mapview）

每個儀表板在地圖交叉比對頁面提供圖層開關，結構如下：

- **組件圖層**（各儀表板自有）：顯示組件空間資料，可單獨切換
- **基本圖層**（所有儀表板通用）：
  - 自行車道、公車站、人行道分布
  - 閒置市有財產、土壤液化潛勢、山坡地風險地點
  - 警察相關設施、緊急避難設施
  - 火災風險地點、火災應變設施
  - 醫療院所分布、自行車道路網圖資

**地圖互動功能**：點位可點擊開啟資訊面板；交通路況以線段顏色（紅/橙/綠）表示壅塞程度。

## 五、AI 對話功能（臺北城市儀表板小幫手）

| 項目 | 說明 |
|------|------|
| 入口 | 所有頁面右下角的圓形 AI 圖示按鈕 |
| 介面 | 浮動對話框，支援文字輸入 |
| 功能1 | 依使用者描述自動推薦並組合儀表板 |
| 功能2 | Tool Calling：呼叫後端 API 查詢組件資料後回覆 |
| AI 模型 | TWCC llama3.3-ffm-70b-16k-chat（OpenAI 相容 API）|
| 向量搜尋 | Qdrant + 本地 ONNX 嵌入模型（e5）進行語意搜尋 |
| 對話紀錄 | 存入 `ai_chatlog` DB 表 |
| API端點 | `POST /api/v1/ai/chat/twai` |
