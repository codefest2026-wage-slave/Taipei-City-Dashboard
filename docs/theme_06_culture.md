## 使用限制與注意事項

1. **訪客模式**: 不需API金鑰，但每IP每日限50次呼叫（基礎服務）
2. **會員模式**: 需要 Client ID + Client Secret，透過OAuth2取得Access Token（有效期1天）
3. **計費方式**: 計次（1,500次/1點）或計量（150MB/1點）
4. **進階服務**: 部分資料（如道路交通VD）需要進階方案才能存取
5. **OData支援**: 所有API支援 `$select`, `$filter`, `$orderby`, `$top`, `$skip` 查詢語法
6. **GeoHash空間查詢**: 支援 `startswith(StopPosition/GeoHash, 'xxxx')` 進行空間篩選
7. **條件快取**: 支援 `If-Modified-Since` header 減少不必要資料傳輸
